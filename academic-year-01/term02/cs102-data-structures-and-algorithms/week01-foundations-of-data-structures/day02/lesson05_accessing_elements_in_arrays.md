### lesson 05: Accessing elements in arrays

1. Index
 
A number that tells you the position of an element in a list. In Python, indices start at 0.
 
2. Valid index range for access
 
For safe access list[i], the index i must satisfy: 0 <= i < len(list).
 
3. Valid index range for insert
 
For safe insert list.insert(i, value), the index i must satisfy: 0 <= i <= len(list).
 
append()
 
A list method that adds a new element to the end of the list.
 
insert(index, value)
 
A list method that adds a new element at a specific index and shifts later elements one position to the right.
