**Slicing using Index**
- List holds charaters ["c", "a", "s", "h"]
- Strings are immutable
- "".join() = "glue list of string into string"
- split() = string to list

**Example**

- Using "".join() 

letters = ["c", "r", "a", "s", "h", "e", "d"]

parts = letters[0:1] + letters[2:5]
print(parts)          # ['c', 'a', 's', 'h']
print("".join(parts)) # cash
