## Context
In Week 01, this calculator was built using:
- variables
- user input
- arithmetic operations
- basic conditional logic

The focus was on **procedural flow** and understanding Python fundamentals.

## Week 02 Goal
In Week 02, the goal is to **refactor and expand** the existing calculator using:

- Functions and scope isolation
- Menu-driven control flow (while / break / continue)
- Early return for validation
- Logical branching and decision tables
- Loop usage (for / while, counters, totals)

No new features are added initially.
Instead, existing logic from Week 01 is reorganized into functions to improve:
- readability
- reusability
- structure
- maintainability
